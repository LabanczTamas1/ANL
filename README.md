# ANL
Thesis project for an ads and leads company.

Instructions for running the webapplication:
-git clone the repository
-cd App
-npm run dev

-cd backend
-node server.js

To run the program you will need NodeJS on your system and a env. file which will connect you to a Redis database.

Content of Frontend env.:
Vite_Backend_Url

Content of Backend env.:
Port_number
Google_Api_Secret_Key
Google_Email_Secret
Redis_Url
